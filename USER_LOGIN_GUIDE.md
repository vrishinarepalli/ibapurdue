## â“ Troubleshooting

### "Invalid email or password"
**Fix:** Double-check your email and password. Try resetting your password if needed.

### "Unauthorized: Not an admin account"
**Fix:** You need to request admin access first (Method 3). Or contact the main admin.

### "Touch ID / Face ID not available on this device"
**Fix:**
- Ensure Touch ID/Face ID is enabled in System Settings
- Use Safari (iPhone/Mac) or Chrome/Edge (Windows)
- Make sure you have at least one fingerprint/face registered

### "No biometric credentials registered"
**Fix:** This is expected! Follow the First-Time Setup steps above.

### "Authentication cancelled or not allowed"
**Fix:** You cancelled the Touch ID prompt. Just click the button again.

### Touch ID works on MacBook but not iPhone
**Fix:** This is normal! You need to register **separately** on each device:
- Register Touch ID on MacBook
- Register Face ID on iPhone
- Both will work independently
