# IBA Purdue

Student-run intramural basketball tournament website for the Indian Basketball Association at Purdue University.

**Site:** https://ibapurdue.online

---

## Stack

- React 18 + Babel (browser-side JSX, no build step)
- Firebase — Firestore, Auth, Cloud Functions, Hosting
- Anthropic Claude for AI-assisted features

## What's here

The site covers registration, scheduling, bracket tracking, standings, and team rosters. There's a restricted admin interface for managing teams, payments, and schedule updates. Cloud Functions handle anything that needs to run server-side.
